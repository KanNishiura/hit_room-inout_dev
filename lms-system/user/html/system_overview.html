システム概要

システムの目的

本システムは、ユーザーがJavaの基本からWeb開発までを体系的に学ぶためのeラーニングプラットフォームです。各ユニットごとに動画コンテンツと課題を通じて理解を深め、定期考査を通じて学習成果を確認することができます。

主な機能

  • ユーザー管理: ログイン、プロフィール編集、パスワード変更
  • コンテンツ管理: ユニットごとの動画視聴、進捗管理
  • 課題管理: 課題の提出とフィードバック
  • 定期考査: 各ユニットごとの定期考査と成績表示
  • 成績管理: ユニットごとの成績の表示と分析

利用方法

  1. ログインします。
  2. 各ユニットの動画を視聴し、課題を提出します。
  3. 定期考査を受験し、成績を確認します。
  4. マイページからプロフィールの編集やパスワードの変更ができます。

システム要件

本システムは、最新のWebブラウザ（Google Chrome、Mozilla Firefox、Microsoft Edge、Safari）での動作を推奨しています。JavaScriptを有効にし、Cookieの使用を許可してください。
